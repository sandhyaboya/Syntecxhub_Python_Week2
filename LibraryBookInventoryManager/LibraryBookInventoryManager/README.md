# Library Book Inventory Manager

A simple Python CLI project to manage library books using JSON storage.

## Features
- Add new books
- Search by title
- Issue or return books
- Generate a summary report
- Data saved in JSON file for persistence

## How to Run
1. Make sure you have Python 3 installed.
2. Open a terminal and navigate to the project folder.
3. Run the program:

```
python main.py
```
